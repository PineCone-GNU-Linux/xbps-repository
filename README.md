# xbps-repository

This is the repository of packages. For PineCone GNU/Linux.

## How to install:

Run this command:
```echo "repository=https://raw.githubusercontent.com/PineCone-GNU-Linux/xbps-repository/main/" | sudo tee /etc/xbps.d/pinecone.conf```

... And update it.
```sudo xbps-install -S```
